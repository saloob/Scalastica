============ Installation ============
1. Create a database (codexworld) at phpMyAdmin and import the db_images.sql file into the database.
2. Open the "db.php" file and change the $dbHost, $dbUsername, $dbPassword, and $dbName variable's value with your database credentials.
4. Now run the "index.php" file and test the drag & drop reorder images functionality.
